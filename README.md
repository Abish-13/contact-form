# Simple Contact Form with JavaScript Validation

This is a clean, simple, and functional contact form built using only HTML, CSS, and vanilla JavaScript. It's a foundational project for any website, featuring client-side validation to ensure that users do not submit empty fields.


## âœ¨ Features

* **Clean UI:** A modern, minimal design that is easy to integrate into any webpage.
* **HTML Structure:** Semantic HTML5 for the form, labels, and inputs.
* **CSS Styling:** Styled with CSS for a responsive, centered layout.
* **Client-Side Validation:** Uses JavaScript to check for empty fields (`name`, `email`, `message`) on submit.
* **User Feedback:** Displays a clear error message if validation fails, preventing the form from being submitted.
